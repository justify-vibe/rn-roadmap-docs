# 学习资源推荐

精选的免费和付费资源，覆盖从入门到进阶的各个阶段。

## 官方文档（首选）

| 资源 | 链接 | 说明 |
|------|------|------|
| React Native 官方文档 | [reactnative.dev](https://reactnative.dev) | 最权威的参考资料 |
| Expo 文档 | [docs.expo.dev](https://docs.expo.dev) | Expo API 和指南 |
| React 官方文档 | [react.dev](https://react.dev) | React 基础必看 |
| React Navigation | [reactnavigation.org](https://reactnavigation.org) | 导航完整指南 |

## JavaScript 基础

### 免费资源

- **[javascript.info](https://javascript.info)** — 最好的 JS 入门教程，有中文版，内容系统全面
- **[freeCodeCamp](https://freecodecamp.org)** — 免费交互式课程，完成后有证书
- **[MDN Web Docs](https://developer.mozilla.org/zh-CN)** — JS 语言参考，查语法必备

### 推荐书籍

- 《JavaScript 权威指南》— 入门到精通
- 《你不知道的 JavaScript》— 深入理解 JS 特性

---

## React 基础

### 免费资源

- **[react.dev/learn](https://react.dev/learn)** — 官方教程，有完整的互动练习
- **[Scrimba React 课程](https://scrimba.com/learn/learnreact)** — 免费互动视频
- **YouTube: Traversy Media** — 搜索 "React Crash Course"

### 练习项目

完成以下项目可以检验 React 基础：

1. 井字棋游戏（跟着官方教程做）
2. Todo List（增删改查）
3. 天气应用（调用公开 API）

## React Native 入门

### 视频教程

- **YouTube: Expo** — [官方 YouTube 频道](https://www.youtube.com/@expo-io)，有大量教程
- **YouTube: William Candillon** — "Can it be done in React Native" 系列，动画进阶
- **YouTube: notJust.dev** — 完整项目教程，适合实战学习

### 实战课程（付费）

- **[Zero to Mastery: React Native](https://zerotomastery.io)** — 系统完整，适合新手
- **[Udemy: Stephen Grider 的 RN 课程](https://udemy.com)** — 口碑很好，经常打折

## 实战练习 API

| API | 用途 | 文档 |

| JSONPlaceholder | 模拟 REST API | [jsonplaceholder.typicode.com](https://jsonplaceholder.typicode.com) |

| PokeAPI | 宝可梦数据 | [pokeapi.co](https://pokeapi.co) |
| NewsAPI | 新闻数据 | [newsapi.org](https://newsapi.org) |

## 设计资源

### UI 设计灵感

- **[Dribbble](https://dribbble.com)** — 搜索 "mobile app UI" 获取设计灵感
- **[Mobbin](https://mobbin.com)** — 真实 App 截图合集，按功能分类
- **[UI8](https://ui8.net)** — 优质 UI 模板（部分付费）

### 图标资源

- **[icons.expo.fyi](https://icons.expo.fyi)** — Expo 内置图标浏览
- **[Phosphor Icons](https://phosphoricons.com)** — 免费精美图标集
- **[Heroicons](https://heroicons.com)** — Tailwind 出品，简洁实用

### 字体资源

- **[Google Fonts](https://fonts.google.com)** — 大量免费字体
- Expo 内置 `expo-font` 支持加载自定义字体
